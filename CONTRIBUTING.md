# Contributing

Your contributions are always welcome!

## Guidelines

- Add one link per pull request.
- Add the link: `- [Tool Name](https://example.com/) - A short description ending with a period.`
- Keep descriptions concise and meaningful.
- Add the link in alphabetical order within its category.
- Check your spelling and grammar.
- Make sure your text editor is set to remove trailing whitespace.
- The pull request and commit should have a useful title.

## Quality Standards

- The tool must be free to use (or have a meaningful free tier).
- The tool should be actively maintained.
- The link must be to a working URL.

Thank you for your suggestions!
